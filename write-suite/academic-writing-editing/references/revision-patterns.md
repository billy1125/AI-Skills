# Academic English Revision Patterns

Use this file when polishing academic English at the sentence or paragraph level. Preserve the user's meaning unless the user explicitly asks for rewriting with stronger intervention.

## Revision priorities

Apply revisions in this order:

1. Meaning preservation
2. Logical clarity
3. Academic tone
4. Concision
5. Fluency
6. Style refinement

Do not sacrifice precision for elegance. Do not add new claims, evidence, citations, or causal interpretations that are not present in the original text.

## Common sentence-level problems and revisions

### Wordy phrasing

Pattern:
- due to the fact that → because
- in order to → to
- it is important to note that → importantly / this suggests / omit if unnecessary
- there is a need to → should / requires

Examples:
- Original: In order to understand the issue, it is important to examine the institutional context.
- Revised: To understand the issue, the institutional context should be examined.

### Informal phrasing

Pattern:
- a lot of → many / substantial / numerous
- get → obtain / receive / become
- look at → examine / investigate / analyze
- deal with → address
- big problem → significant challenge

Examples:
- Original: Many researchers have looked at this problem.
- Revised: Many researchers have examined this issue.

### Vague reference

Pattern:
- this / it / they without a clear referent → repeat or specify the noun.

Examples:
- Original: This shows that the policy was effective.
- Revised: This pattern suggests that the policy may have been effective.

### Overly strong claims

Pattern:
- proves → suggests / indicates / provides evidence that
- always → often / generally / in many cases
- causes → is associated with / may contribute to

Examples:
- Original: The results prove that feedback causes higher achievement.
- Revised: The results suggest that feedback is associated with higher achievement.

### Weak topic sentence

Pattern:
- Background-only opening → claim-oriented opening.

Examples:
- Original: Student engagement has been widely discussed in higher education research.
- Revised: Student engagement is a central factor in higher education research because it is closely linked to persistence and academic performance.

### Evidence without interpretation

Pattern:
- Add an explanation of why the evidence matters.

Examples:
- Original: Smith (2020) found that feedback improved student motivation.
- Revised: Smith (2020) found that feedback improved student motivation, suggesting that instructional support may shape students' engagement with academic tasks.

## Paragraph-level revision patterns

### Clarifying the main claim

Use when the paragraph contains relevant information but lacks a visible argument.

Template:
- This paragraph argues that [main claim]. It first shows [evidence 1], then explains [evidence 2], and finally suggests [implication].

Revision move:
- Add or revise the first sentence so it names the paragraph's analytical purpose.

### Improving flow

Use when the paragraph has good content but weak transitions.

Revision moves:
- Add a transition only when the relation is unclear.
- Repeat key terms strategically.
- Use this / these + noun to connect sentences.

Examples:
- This pattern suggests...
- These findings indicate...
- This limitation is important because...
- Such variation may reflect...

### Tightening academic style

Use when the prose is understandable but too loose.

Revision moves:
- Replace informal verbs with analytical verbs.
- Reduce redundant clauses.
- Use nominal phrases only when they improve precision.
- Maintain subject-verb clarity.

## Polishing levels

### Light polish

Use when the user wants minimal editing.

Allowed changes:
- grammar
- word choice
- punctuation
- minor flow improvements

Avoid:
- restructuring the paragraph
- changing argument order
- adding interpretation

### Moderate polish

Use when the paragraph is understandable but stylistically weak.

Allowed changes:
- sentence restructuring
- improved transitions
- stronger academic phrasing
- clearer topic sentence

Avoid:
- adding unsupported claims
- changing the intended argument

### Substantive revision

Use when the paragraph has structural or logical problems.

Allowed changes:
- reorder sentences
- split long sentences
- add signposting
- make implicit logic explicit

Flag the change if it substantially reorganizes the paragraph.

## Useful academic verbs

For literature review:
- argues
- suggests
- demonstrates
- examines
- identifies
- highlights
- challenges
- extends

For results:
- indicates
- reveals
- shows
- suggests
- predicts
- is associated with
- remains significant
- varies across

For discussion:
- implies
- may reflect
- may explain
- is consistent with
- contributes to
- raises questions about
- provides evidence for

## Avoid list

Avoid these unless context requires them:
- obviously
- clearly, when unsupported
- very
- really
- a lot
- prove, unless proof is logically or mathematically warranted
- perfect
- totally
- huge
