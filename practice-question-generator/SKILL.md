# 練習題產生器

使用此技能將教材轉換為學生練習題。僅接受以下輸入來源：上傳的 Word 檔案、上傳的 PowerPoint 檔案、上傳的 PDF 檔案或貼上的網頁連結。除非使用者明確透過上傳或 URL 提供內容，否則請勿使用 Google 雲端硬碟或其他連接器。

## 輸入規則

1. 識別源材料。

- 對於上傳的 Word、PowerPoint 或 PDF 文件，請從上傳的文件中提取教學內容。

- 對於貼上的網頁鏈接，如果可以訪問，請讀取頁面內容。

- 如果無法存取相關內容，請要求使用者上傳文件或貼上相關文字。

2. 產生問題前，詢問使用者所需的題型，除非使用者已指定。

支援的問題類型：

- 選擇題：單一選擇題。

- 複選擇題：多選題，允許有多個正確答案（視情況而定）。

- 簡答題：簡答題，要求使用者用簡潔的文字作答。

- 整合題：包含選擇題、複選擇題和簡答題的平衡混合題集。

3. 若使用者僅提及「練習題」、「題目」、「測驗題」、「複題」等廣泛的詞語，而未明確指定題型：

- 如果使用者似乎在詢問配置或規劃，請詢問是否需要選擇題、複選擇題、簡單答題或整合題。

- 如果使用者明確希望立即獲得結果，則預設為整合題。

4. 在產生最終輸出之前，詢問使用者所需的輸出檔案格式，除非使用者已指定。

支援的格式：

- 純文字

- Markdown

- HTML

- Word

- PowerPoint

5. 如果使用者指定了一種或多種題型，則僅產生這些題型。不要為了完整性而添加其他題型。

6. 如果使用者指定了一種或多種檔案格式，則僅提供這些格式。如果需要建立文件，請遵循環境中提供的相關文件、投影片或文件產生說明。

## 問題產生流程

1. 閱讀教材，並依主題、章節、投影片部分、標題或概念單元進行分段。

2. 確定材料中隱含的學習目標。

3. 在材料支援的情況下，產生涵蓋定義、概念理解、應用、比較、解釋和綜合等方面的問題。

4. 避免捏造資料中未提及的事實。若需要外部知識，請明確標註或省略該問題。

5. 保持語言與原文一致。如果使用者使用繁體中文作答，則預設使用繁體中文。

6. 除非使用者另有說明，否則問題難度應與原文相符。盡量均衡地分佈基礎題、中級題和應用題。

## 輸出標準

### 必要條件

- 必須先生成問題，然後再生成答案文件。

- 題目之下不能出現答案，答案最後生成分別顯示或分成兩個檔案，以確保學生在嘗試回答問題之前不會看到答案。

### 問題規格

對於每個問題，請包括：
- 題號
- 題型
- 題目
- 選項，僅適用於選擇題和複選題

### 答案規格

答案最後生成分別顯示或分成兩個檔案，並且對於每個問題，請包括：
- 正確答案
- 解析
- 對應教材重點或來源概念

### 題型特定規則

對於選擇題：
- 預設提供四個選項，標記為 A-D。
- 僅使用一個正確答案。
- 根據材料，使干擾因素看似合理，但毫無疑問是錯誤的。

對於複選題：
- 預設提供五個選項，標記為 A-E。
- 如果資料支持這一點，請包括兩個或多個正確答案。
- 說明該問題可能有多個正確答案。

對於簡答題：
- 提供簡潔的參考答案。
- 包括有用的評分點或預期關鍵字。

對於整合問題：
- 使用平衡的混合，除非使用者指定計數。
- 預設混合：40% 選擇題、30% 複選題、30% 簡答題。
- 如果要求的問題數量較少，請保留多樣性而不是精確的百分比。
