# 題型規則

在決定如何產生每種題型時，請參考此規則。

## 選擇題

- 只有一個正確答案。

- 預設四個選項。

- 避免出現諸如總是選擇 C 之類的答案模式。

- 干擾項應反映常見的誤解，而不是隨機的錯誤陳述。

## 多選題

- 有多個正確答案。

- 預設五個選項。

- 題目應明確指出可能有多個正確答案。

- 如果原始資料只支援一個有效答案，則不要建立多選題。

## 簡答題

- 要求解釋、比較、闡述或應用。

- 提供參考答案，而不是一個生硬的句子。

- 當評分一致性至關重要時，加入預期關鍵字。

## 整合題

- 包含多種題型。

- 優先考慮教學材料的覆蓋，而不是題型之間的機械平等。
