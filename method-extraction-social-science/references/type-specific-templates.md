# 類型化欄位模板

本文件用於在完成通用 method extraction 後，依研究類型補充專屬欄位。

## 一、問卷研究（survey）
建議補充欄位：
- respondent type
- scale format
- questionnaire source
- EFA details
- CFA details
- reliability indicators
- convergent validity
- discriminant validity
- common method bias / variance checks
- SEM / PLS-SEM / regression path testing
- mediation analysis
- moderation analysis
- multigroup analysis

## 二、實驗研究（experiment）
建議補充欄位：
- treatment / manipulation
- control condition
- assignment mechanism
- randomization
- manipulation check
- pretest / posttest structure
- dependent variable timing
- covariates
- laboratory / field / online setting
- ANOVA / ANCOVA / regression specification

## 三、準實驗研究（quasi-experiment）
建議補充欄位：
- treatment definition
- comparison group
- assignment mechanism
- identification strategy
- treatment timing
- pre-period and post-period structure
- balance checks
- pre-trend checks
- endogeneity handling
- placebo tests
- sensitivity checks
- robustness checks

## 四、縱貫或面板研究（longitudinal / panel）
建議補充欄位：
- wave structure
- time interval
- attrition handling
- lag design
- fixed effects / random effects
- within-between logic

## 五、多層次研究（multilevel study）
建議補充欄位：
- nested structure
- level-1 variables
- level-2 variables
- level-3 variables if applicable
- aggregation logic
- ICC or multilevel justification
- centering strategy
- HLM / MLM estimation details

## 六、多研究論文（multi-study paper）
若論文包含 Study 1、Study 2、Study 3 等：
1. 先逐一抽取每個 study 的完整 schema。
2. 每個 study 使用相同欄位結構。
3. 最後再做 integrated comparison note，說明各 study 之間的設計差異、測量差異、分析差異與互補性。
