# 標準化方法標籤對照

本文件用於統一 method extraction 的欄位名稱與方法標籤，避免不同論文使用不同說法而降低 comparison 能力。

## 一、研究設計（study design）
優先使用以下標準標籤：
- survey
- longitudinal survey
- experiment
- quasi-experiment
- archival / secondary data
- panel data
- multilevel study
- mixed methods
- qualitative coding with quantitative validation

## 二、常見統計分析方法（analysis methods）
優先使用以下標準標籤：
- regression
- multiple regression
- logistic regression
- hierarchical regression
- structural equation modeling (SEM)
- partial least squares SEM (PLS-SEM)
- confirmatory factor analysis (CFA)
- exploratory factor analysis (EFA)
- analysis of variance (ANOVA)
- analysis of covariance (ANCOVA)
- hierarchical linear modeling (HLM)
- multilevel modeling (MLM)
- difference-in-differences (DID)
- fixed effects model
- random effects model
- mediation analysis
- moderation analysis
- multigroup analysis
- propensity score matching (PSM)
- instrumental variable (IV)

## 三、量測品質相關術語
優先使用以下標準標籤：
- reliability
- Cronbach's alpha
- composite reliability (CR)
- convergent validity
- discriminant validity
- average variance extracted (AVE)
- common method bias (CMB)
- common method variance (CMV)
- manipulation check
- measurement invariance

## 四、品質判讀標籤
評等統一使用：
- high
- medium
- low
- unclear

## 五、缺漏資訊標記
若論文未清楚說明，統一使用：
- not reported
- unclear
- inferred with low confidence
