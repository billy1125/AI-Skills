# Analysis Framework

## Minimal field set

當資訊有限時，至少保留以下欄位：
- 題名
- 作者與年份
- 出處
- 研究主題
- 研究問題
- 核心論點
- 方法
- 主要發現
- 研究缺口
- 信心等級

## Research gap classification

### 理論缺口
判斷線索：
- 理論框架缺乏整合
- 只引用理論但沒有真正檢驗
- 缺少替代理論比較
- 新現象沒有足夠理論支撐

### 方法缺口
判斷線索：
- 研究設計過度單一
- 缺少縱貫、實驗或混合方法
- 量測方式有侷限
- 過度依賴自陳資料或單一資料庫

### 樣本缺口
判斷線索：
- 样本集中於特定年齡、地區、產業、平台或族群
- 樣本規模不足
- 缺少跨族群比較

### 情境缺口
判斷線索：
- 結論只在特定產業、文化、制度或平台成立
- 不同情境的可轉移性未檢驗

### 時間缺口
判斷線索：
- 只有橫斷面資料
- 缺少長期追蹤
- 未處理前後時期差異

### 變數缺口
判斷線索：
- 缺少中介、調節、邊界條件
- 變數定義不一致
- 可能存在反向因果或遺漏機制

## Suggestion templates

### 未來研究方向模板
- 可進一步檢驗 [理論/機制] 在 [新情境/族群] 中是否仍成立。
- 可採用 [縱貫/實驗/混合方法] 補強目前以 [橫斷/單一方法] 為主的證據。
- 可比較 [情境 A] 與 [情境 B] 的差異，以釐清邊界條件。
- 可納入 [中介/調節變數] 以解釋現有研究結果不一致之處。

### 文獻回顧段落模板
1. 先界定研究主題與核心問題。
2. 再整理主要理論與研究脈絡。
3. 接著比較不同研究方法與主要發現。
4. 最後指出研究缺口並引出你的研究動機。

## Confidence language

### 高信心
可用較確定語氣，但仍避免過度延伸。

### 中信心
加上限定語，例如：
- 依可讀內容判斷
- 目前資訊顯示
- 就已取得頁面而言

### 低信心
明確使用保留語氣，例如：
- 初步判斷
- 僅能根據摘要推估
- 完整結論仍需全文確認
